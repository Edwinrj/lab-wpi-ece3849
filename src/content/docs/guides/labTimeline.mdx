# ECE3849 Lab Timeline & Summaries

<Aside type="note" title="Overview">
  Each lab in **ECE3849: Real-Time Embedded Systems** builds upon the previous one,
  culminating in a complete FreeRTOS-based multitasking system that integrates sensing,
  signal processing, and interactive gameplay.
</Aside>

---

## Lab Schedule & Summaries

<CardGrid stagger>

  <Card title="Lab 0 – Stopwatch & Graphical Menu" icon="seti:clock">
    <Badge text="Due Mar 26" variant="success" size="medium" />
    <p><strong>Objective:</strong> Familiarize with the TM4C1294XL LaunchPad, Code Composer Studio,
    and basic peripheral programming.</p>
    <p><strong>Highlights:</strong> System clock setup, LCD initialization, timer ISR, button debouncing,
    and menu navigation.</p>
  <LinkButton href="/labs/lab0/" target="_blank">View Lab 0 →</LinkButton>
  </Card>

  <Card title="Lab 1 – FreeRTOS Stopwatch" icon="seti:audio">
    <Badge text="Due Apr 16" variant="success" size="medium" />
    <p><strong>Objective:</strong> Move the stopwatch from a bare-metal super-loop into a FreeRTOS application built from scratch.</p>
    <p><strong>Highlights:</strong> Manual FreeRTOS integration, task creation, queues, periodic scheduling, and RTOS-based stopwatch design.</p>
  <LinkButton href="/labs/lab1/" target="_blank">View Lab 1 →</LinkButton>
  </Card>

  <Card title="Lab 2 – Multi-Screen Dashboard" icon="document">
    <Badge text="Due Apr 23" variant="success" size="medium" />
    <p><strong>Objective:</strong> Refactor the Lab 1 stopwatch into a modular multi-screen application with interrupt-driven buttons, joystick navigation, and microphone level monitoring.</p>
    <p><strong>Highlights:</strong> Multiple source files, binary semaphores, software timers, ADC microphone sampling, and true multitasking across screens.</p>
  <LinkButton href="/labs/lab2/" target="_blank">View Lab 2 →</LinkButton>
  </Card>

  <Card title="Lab 3 – Snake Game" icon="puzzle">
    <Badge text="Due Apr 30" variant="success" size="medium" />
    <p><strong>Objective:</strong> Complete a Snake game on FreeRTOS by adding game mechanics, synchronization, and event-driven coordination between tasks.</p>
    <p><strong>Highlights:</strong> Mutex for shared game state, task notifications for event-driven rendering, event group for FOOD_EATEN and GAME_OVER audio feedback, collision detection, score and speed scaling.</p>
  <LinkButton href="/labs/lab3/" target="_blank">View Lab 3 →</LinkButton>
  </Card>

  <Card title="Lab 4 – Snake Game Part 2" icon="rocket">
    <small>Due to Thanksgiving break, there is no rush on Nov 25.</small>
    <p><strong>Objective:</strong> Enhance your Snake game with advanced FreeRTOS timing, synchronization, and performance analysis: precise periodic tasks, buffered input, real-time metrics, and robustness.</p>
    <p><strong>Highlights:</strong> Hardware timer jitter measurement; vTaskDelay vs vTaskDelayUntil comparison; counting semaphore joystick buffer; 10ms chronometer (software timer); wall collisions & game-over animation; stack usage display, optimization & overflow detection.</p>
  </Card>

  <Card title="Lab 5 – Final System Integration" icon="laptop">
    <p><strong>Objective:</strong> Integrate all previous modules into a single multitasking system.</p>
    <p><strong>Highlights:</strong> FreeRTOS task coordination, inter-task communication, performance monitor,
    and adaptive Snake gameplay.</p>
  </Card>

</CardGrid>

# ECE3849 Lab Timeline & Summaries

<Aside type="note" title="Overview">
  Each lab in **ECE3849: Real-Time Embedded Systems** builds upon the previous one,
  culminating in a complete FreeRTOS-based multitasking system that integrates sensing,
  signal processing, and interactive gameplay.
</Aside>

## Lab Schedule & Summaries

    <p><strong>Objective:</strong> Interface and read light, temperature, and microphone sensors in real time.</p>
    <p><strong>Highlights:</strong> ADC/I2C interfacing, ISR-driven sampling, PWM feedback,

    <LinkButton href="/labs/lab1/">View Lab 1 →</LinkButton>

  <Card title="Lab 2 – 8-Band Spectrum Analyzer" icon="seti:audio">
    <Badge text="Due Nov 12" variant="note" size="medium" />
    <p><strong>Objective:</strong> Implement a real-time audio spectrum analyzer using FFT on sampled audio.</p>
    <p><strong>Highlights:</strong> ADC + DMA, circular buffers, 256-point FFT, dB conversion,
    and performance measurement.</p>
    <LinkButton href="/labs/lab2/">View Lab 2 →</LinkButton>
  </Card>

  <Card title="Lab 3 – Migration to FreeRTOS" icon="document">

    <p><strong>Objective:</strong> Migrate your application to FreeRTOS and explore multitasking concepts.</p>
    <p><strong>Highlights:</strong> Task creation, semaphores, message queues, priority tuning,
    and runtime profiling.</p>
    <LinkButton href="/labs/lab3/">View Lab 3 →</LinkButton>
  </Card>

  <Card title="Lab 4 – RTOS Snake Game" icon="rocket">
    <Badge text="Due Dec 1" variant="caution" size="medium" />
    <small>Due to Thanksgiving break, there is no rush on Nov 25.</small>
    <p><strong>Objective:</strong> Create an interactive Snake game using FreeRTOS multitasking.</p>
    <p><strong>Highlights:</strong> Input handling, game logic, software timers, LCD mutex, and sound feedback.</p>
    <LinkButton href="/labs/lab4/">View Lab 4 →</LinkButton>
  </Card>

  <Card title="Lab 5 – Final System Integration" icon="laptop">
    <Badge text="Due Dec 10" variant="danger" size="medium" />

    <p><strong>Highlights:</strong> FreeRTOS task coordination, inter-task communication, performance monitor,

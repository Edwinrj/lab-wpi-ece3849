## Configuring the EKâ€‘TM4C1294XL Board

<Card title="Connect Your LaunchPad" icon="usb">
Plug your **EK-TM4C1294XL** into the computer using the supplied USB cable. CCS should automatically detect the ICDI debug interface.
</Card>

![Board Connection](/public/assets/ccs_install/ek_tm4c1294xl_connection.png)

### Create a New Project

1. **File â†’ New â†’ CCS Project**
2. Set the following parameters:
   - **Target:** `TM4C1294NCPDT`
   - **Connection:** `Stellaris ICDI`
   - **Project name:** `TM4C_Blink`
3. Finish and open the generated `main.c`.

![New Project Wizard](/public/assets/ccs_install/ccs_new_project.png)

### Example Blink Test

```c
#include <stdint.h>
#include <stdbool.h>
#include "inc/hw_memmap.h"
#include "driverlib/sysctl.h"
#include "driverlib/gpio.h"

int main(void) {
    SysCtlClockFreqSet((SYSCTL_XTAL_25MHZ | SYSCTL_OSC_MAIN |
                        SYSCTL_USE_PLL | SYSCTL_CFG_VCO_480), 120000000);

    SysCtlPeripheralEnable(SYSCTL_PERIPH_GPION);
    GPIOPinTypeGPIOOutput(GPIO_PORTN_BASE, GPIO_PIN_0);

    while (1) {
        GPIOPinWrite(GPIO_PORTN_BASE, GPIO_PIN_0, GPIO_PIN_0);
        SysCtlDelay(20000000);
        GPIOPinWrite(GPIO_PORTN_BASE, GPIO_PIN_0, 0);
        SysCtlDelay(20000000);
    }
}
```

## Installing TivaWare

<LinkCard
  title="Download TivaWare for TM4C"
  description="Official TI driver library for TM4C microcontrollers"
  href="https://www.ti.com/tool/SW-TM4C"
/>

### Installation Steps

1. Extract TivaWare archive to:
   - `C:\ti\TivaWare_C_Series-2.2.0.295` (Windows)
   - `/opt/ti/TivaWare_C_Series-2.2.0.295` (Linux/macOS)
2. In CCS â†’ **Project Properties â†’ Build â†’ Include Options**, add:
   ```
   C:\ti\TivaWare_C_Series-2.2.0.295\driverlib\inc
   ```
3. Under **Linker â†’ File Search Path**, add:
   ```
   C:\ti\TivaWare_C_Series-2.2.0.295\driverlib\ccs\Debug\driverlib.lib
   ```

![TivaWare Configuration](/public/assets/ccs_install/ccs_tivaware_config.png)

## Troubleshooting

<Tabs>
<TabItem label="USB Connection Issues">
- Ensure ICDI drivers are installed (Windows)
- On Linux, verify device recognition:
```bash
lsusb | grep -i texas
```
- Try different USB ports or cables
![USB Issues](/public/assets/ccs_install/ccs_troubleshoot_usb.png)
</TabItem>

<TabItem label="Build Errors">
- Check include and library paths
- Ensure correct target (`TM4C1294NCPDT`) selected
- Clean and rebuild project
![Build Errors](/public/assets/ccs_install/ccs_troubleshoot_build.png)
</TabItem>
</Tabs>

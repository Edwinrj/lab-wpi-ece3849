### ðŸ§° Install Code Composer Studio (CCS 12.8.1)
<Aside type="tip" title="Why CCS 12.8.1?">
  <div>
    <p><strong>CCS 12.8.1</strong> is recommended for its stability and compatibility with the TM4C1294XL LaunchPad.</p>
    <ul>
      <li>Based on Eclipse with advanced debugging features</li>
      <li>Newer <strong>CCS 20</strong> uses Theia (VS Code-like), which may affect legacy SDKs</li>
    </ul>
  </div>
</Aside>

1. Navigate to the downloaded folder and run:
   ```bash
   ccs_setup_12.8.1.00005.exe
   ```
2. When prompted for an installation directory, set the path to:
   ```bash
   C:\ti\ccs1281
   ```
3. Choose **Custom Installation** when asked for setup type.

<p align="center">
  <img src="https://hackmd.io/_uploads/rkP_X74Axx.png" alt="CCS Custom Installation" width="450" />
  <br />
  <sub>Select Custom Installation during setup</sub>
</p>

4. In the list of supported processors, check:
   > **TM4C12X ARM Cortex M4F core-based MCUs**
5. Continue with the default options for the rest of the installation.

<p align="center">
  <img src="https://hackmd.io/_uploads/Hksj77N0le.png" alt="Select TM4C12X ARM Cortex M4F core-based MCUs" width="450" />
  <br />
  <sub>Select TM4C12X ARM Cortex M4F core-based MCUs</sub>
</p>

<Aside type="tip" title="Tip">
ðŸ’¡ After installation, verify that CCS created the folder `C:\ti\ccs1281`.
</Aside>

### ðŸ§© Install TivaWare Drivers

1. Extract the package:
   ```bash

   ```


### ðŸ§° Install Code Composer Studio (CCS 12.8.1)

1. Make installer executable and run:
   ```bash
   chmod +x ccs_setup_12.8.1.00005.bin
   sudo ./ccs_setup_12.8.1.00005.bin
   ```
2. Set installation directory:
   ```bash
   /opt/ti/ccs1281
   ```
3. Select:
   > TM4C12X ARM Cortex M4F core-based MCUs
4. Add user to `dialout` for USB debugging:
   ```bash
   sudo usermod -a -G dialout $USER
   ```
5. Launch CCS:
   ```bash
   /opt/ti/ccs1281/ccstudio
   ```
